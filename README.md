# sort_report

### 這個是作為大一電機工程學系計算機概論作業
### 網址: https://allencreeperqq.github.io/sort_report/

## 版本歷程

### v0.1.0 - 2026-03-09

- 來源：changed.txt 舊紀錄。
- 將首頁 index.html 推送更新。

### v0.2.0 - 2026-03-16

- 更新首頁 index.html 主題為「演算法學習網站」。
- 在首頁新增 search.html 入口按鈕，保留原本 sort-html.html 入口。

### v0.3.0 - 2026-03-16

- 新增 search.html 搜尋法學習頁面。
- 內容包含：線性搜尋、二分搜尋、跳躍搜尋。
- 加入與排序頁相同風格的互動機制：
  - 重新產生資料
  - 開始播放 / 暫停
  - 下一步 / 上一步 / 重置
  - 步驟說明、複雜度資訊、C++/Python 程式碼區塊與一鍵複製

### v0.3.1 - 2026-03-16

- 調整 search.html 目標值邏輯：
  - 初始化時，目標值預設為「已生成資料中的值」。
  - 每次重新產生資料時，也會自動挑選「已生成資料中的值」作為目標。

### v0.3.2 - 2026-03-16

- 依桌機觀看需求調整 search.html 版面：
  - 搜尋法選單、資料數量、動畫速度、目標值與所有播放按鈕，全部集中到動畫框正上方。
  - 移除左側固定控制面板，提升電腦版瀏覽與操作可讀性。

### v0.3.3 - 2026-03-16

- 更新首頁 index.html 按鈕文字：
  - 學習排序法
  - 學習搜尋法

### v0.3.4 - 2026-03-16

- 依使用需求再次調整 search.html 版面：
  - 重新加入左側活動視窗（資料數量、動畫速度、目標值）。
  - 將搜尋法選單與播放控制按鈕區移到更靠近動畫框的位置。
  - 與時間複雜度、定義、核心想法、歷史區塊進行版面對調。

### v0.3.5 - 2026-03-16

- 新增首頁 index.html 按鈕滑鼠懸停效果：
  - 滑鼠移到按鈕上方時改為彩色漸層。
  - 加入邊框亮化與彩色光暈陰影。

### v0.4.0 - 2026-03-16

- 視覺與互動優化（search.html、sort-html.html）：
  - favicon 改為 emoji 💫。
  - 區塊上下間距加大，降低內容擁擠感。
  - 背景改為 fixed，捲動時僅內容移動。
  - 毛玻璃面板提高可讀性（降低模糊、提高不透明度）。
  - 左側浮動窗修正內外框定位，避免內框超出外框。
  - 新增左側浮動窗滑動慣性效果。
- 互動按鈕強化：
  - search.html 與 sort-html.html 的「開始播放」按鈕改為高辨識主按鈕樣式。
- 內容同步：
  - self.html 與 self-data.txt 內容同步更新。

### v0.4.1 - 2026-03-16

- 版面順序一致化（sort-html.html）：
  - 將排序頁的「控制按鈕區（開始播放等）」與「時間複雜度 / 演算法定義 / 核心想法 / 歷史背景」區塊對調。
  - 調整後區塊順序與 search.html 一致，閱讀與操作流程更統一。

### v0.4.2 - 2026-03-16

- search.html 新增報告快速存取按鈕：
  - 在頁面上方導覽列加入「查看搜尋報告 PDF」按鈕。
  - 可直接開啟 report of searching.pdf（新分頁）。
